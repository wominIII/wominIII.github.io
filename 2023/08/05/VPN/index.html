Clash 是由 Dreamacro 开发的，是一个使用 Go 开发的、基于规则的隧道，无需服务器和虚拟主机。

在中国大陆地区，Clash 可用于突破防火长城的限制。

2020 年末，Dreamacro 在 GitHub 仓库释出 Clash 的衍生版本 Premium，但并未提供其原始码。

Clash for Windows 是一个运行在 Windows 上的图形化 Clash 分支。它通过 Clash API 来配置和控制 Clash 核心程序，为用户提供可视化的操作界面和更便捷的使用体验。

Clash 是一款出色的代理客户端软件，支持多种代理协议，如 v2ray、Shadowsocks、ShadowsocksR 等。这些协议是服务端的实现，例如 v2ray 使用的是 VMess 协议，Shadowsocks 使用的是 SS 协议。与此不同，VPN 是一个虚拟专用网络，两者是完全不同的概念。

Clash 无需服务器、虚拟主机或域名，它是一个网络代理协议的客户端，可在电脑、安卓和苹果等平台上使用。
